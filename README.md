# indiefinger-cli
cli tool for indiefinger, a webfinger superset

## usage
```console
$ indiefinger [user@]host

        example:
                indiefinger www.w3c.org
```


## api


## installation

    $ npm install -g indiefinger-cli


## running the tests

From package root:

    $ npm install
    $ npm test
